# Workflows

## What is a Workflow?

A sequence of agent steps executed to fulfill a goal.

---

## Example: Viral Content Workflow

Steps:
1. interpret goal
2. generate pillars
3. generate hooks
4. write scripts
5. write captions
6. assign schedule

---

## Example: Affiliate Workflow

Steps:
1. interpret goal
2. generate niche structure
3. build keyword clusters
4. create article briefs
5. generate articles
6. queue publishing

---

## Execution Modes

- sequential
- parallel
- conditional

---

## Status Tracking

Each step:
- pending
- running
- failed
- completed

---

## Retry System

- automatic retries
- manual retry trigger
- failure logs

---

## Adding Workflow

1. define workflow config
2. assign agents
3. define step order
4. register workflow pack
